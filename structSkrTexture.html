Detailed Description

Texture object used by the rendering engine.

Encapsulates GPU texture data and the raw image source from which it was created. The texture may represent diffuse color, normals, specular, or other material properties (see SkrTextureType).

◆ Source

char* SkrTexture::Source

Raw image data in memory.

Pointer to pixel data (e.g. from stbi_load). Used to upload the texture to the GPU. The format (channels, bit depth, layout) depends on the loader. May be NULL once the texture has been uploaded and the CPU copy freed.
