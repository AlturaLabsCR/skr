Use this macro instead of calling 'm_skr_last_error_set_with_meta()' directly. It automatically includes the source file, line number, and function name in the error message prefix.

@usage m_skr_last_error_set("Failed to load texture: %s", path);

◆ MAX_BONE_INFLUENCE

#define MAX_BONE_INFLUENCE   4

Maximum number of bone influences per vertex.

Each vertex can be affected by up to this many bones during skeletal animation. Commonly set to 4, since most real-time rendering pipelines balance flexibility with performance by limiting to four weights per vertex.
